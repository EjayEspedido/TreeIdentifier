Barangay name

Population count

Geographic coordinates

4. Tree Planting Computation Logic

4.1 Tree Quantity Calculation

Calculate recommended number of trees based on:

Area (square meters)

Standard tree spacing rules (configurable)



4.2 Tree Compatibility & Selection

The app must recommend tree species that are:

Native or suitable to Philippine conditions

Appropriate based on:

Urban density

Available planting space

Population pressure

5.1 Built Environment Detection

5.2 Flood Risk Assessment

If area is flood-prone:

Recommend flood-resilient tree species

Exclude unsuitable trees
